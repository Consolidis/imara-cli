IMARA CLI — Product Vision

PROPOSITION DE VALEUR :
Agent de codage agentique accessible via CLI utilisant une API distante.
L'objectif est de fournir un agent solide, simple et flexible permettant
même a un non-developpeur d'utiliser la ligne de commande pour coder efficacement.

PUBLIC CIBLE :
Developpeurs principalement, mais egalement utilisateurs non-techniques
souhaitant un outil CLI puissant et accessible.

FONCTIONNALITES CLES :
- Experience utilisateur interactive et fluide (UI TUI, workflow guide)
- Gestion intelligente du contexte : economie de tokens, longues sessions,
  support des gros projets sans perte de coherence
- Extensibilite via outils mis a disposition du LLM
- Robustesse et stabilite au niveau des standards (Claude Code, Gemini CLI,
  kimi Code CLI)

OBJECTIFS IMMEDIATS :
- Stabiliser la base de code existante
- Implementer les features critiques manquantes (contexte, flux interactif)
- Produire un MVP robuste et competitif

ROADMAP CONDUCTOR :
1. Track Core — Stabilisation de l'architecture et robustesse
2. Track UX   — Experience utilisateur interactive et accessibilite
3. Track Intelligence — Gestion optimale du contexte et economie tokens
