Track 006 — Onboarding & Premiere Experience Utilisateur

Statut : En cours
Cree : 2026-05-17

Objectif
Creer un flux d'onboarding clair et automatisme pour guider l'utilisateur a travers sa premiere configuration (cle API, modele par defaut, workspace). Afficher un tutoriel interactif integre dans le REPL lors du premier lancement.

Scope
- Detection du premier lancement (fichier de config vide ou inexistant).
- Wizard interactif dans le terminal pour configurer cle API et modele.
- Sauvegarde automatique de la configuration initiale.
- Tutoriel de 3 etapes dans le REPL (/tutorial, /start).
- Commande /welcome pour reafficher l'onboarding a tout moment.
