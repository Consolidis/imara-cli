# 🔄 Conductor Workflow

## 1. Inquiry
- Ask clarifying questions.
- Understand scope and constraints.

## 2. Planning
- Create `spec.md` (Technical Specification).
- Create `plan.md` (Step-by-step tasks).
- Get approval.

## 3. Execution (TDD)
- Write tests.
- Implement code.
- Refactor.

## 4. Completion
- Final verification.
- Archive or clean up track.
